# Batch

## Reserved words

```text
shift : Pop the current argument in order to for example treat the following arguments.

goto : It transfers execution to the label specified; execution continues from that point.

call : It also transfers execution to the label but when the processing reaches an exit or end-of-physical-file, execution is transferred back to the instruction directly after the call instruction.
```
