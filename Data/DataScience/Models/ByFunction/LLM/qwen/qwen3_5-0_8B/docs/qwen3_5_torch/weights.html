Module Docstring

Safetensors weight loader for Qwen3.5 checkpoints.

Pure-bookkeeping module — no math lives here. It walks the *.safetensors shards listed in model.safetensors.index.json (or a single-shard file), maps each checkpoint key to a model key via _remap_key, and copies tensors into model.state_dict() with optional dtype casting.

Key things the remap handles:

  • HuggingFace checkpoints store the conditional-generation model with prefix model.language_model.* and model.visual.*. In text-only mode we strip the model.language_model. prefix and skip visual keys.
  • MTP (Multi-Token-Prediction) checkpoints embed auxiliary heads prefixed with mtp.*; those are always skipped.
  • Tied lm_head — if the checkpoint lacks lm_head.weight because the original model tied it to embed_tokens, we copy the embedding across after loading.
